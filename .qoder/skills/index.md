# Superpowers Skills Index
Доступные навыки. Агент выбирает нужные перед написанием кода.

| Навык | Файл | Когда применять |
|-------|------|----------------|
| Аутентификация API | api-authentication.md | При работе с JWT, HMAC, ключами, заголовками |
| UI Компоненты | ui-components.md | При генерации виджетов, тем, анимаций, состояний экрана |

## Правила использования
1. Перед генерацией кода прочти этот индекс.
2. Выбери навыки, релевантные текущей задаче.
3. Загрузи их содержимое в контекст.
4. Строго следуй инструкциям из выбранных файлов.
5. Если задача не покрывается навыками — сообщи об этом и предложи создать новый.
