# Skill: UI Components
## Контекст
Генерация интерфейсов в стиле Linear/Notion для Flutter.

## Правила
1. Строго следуй DESIGN.md в корне проекта.
2. Используй константы из lib/core/theme/ для цветов, отступов, шрифтов.
3. Минимализм: максимум 2 действия на карточку, много воздуха.
4. Все состояния экрана (idle, loading, success, error) должны быть явно покрыты.
5. Анимации только плавные (200ms), без резких скачков.
6. Предпочитай Composition over Inheritance для переиспользуемых виджетов.
7. Не смешивай бизнес-логику и UI. Выносы вызовы API в отдельные классы.
